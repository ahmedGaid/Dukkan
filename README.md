# Dukkan — دكان

Two-sided grocery marketplace for Egypt. Customers order from neighborhood shops (دكاكين);
shop owners manage catalog and orders. One Flutter app, two roles. Arabic-first, RTL-first.

- **Stack:** Flutter · BLoC · Clean Architecture · Firebase (Auth/Firestore/Storage/FCM)
